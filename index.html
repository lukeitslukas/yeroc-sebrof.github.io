Path Finding

For learning C++ in 2nd year of Ethical Hacking I made an attempt at creating path finding algorithms. I wanted to compare Dijkstras algorithm to Lee algorithm. By seeding purely random maps to traverse and giving them to both algorithms each time this eliminated any bias from the rand function in C++ or by the writer. For this I received the grade: B (3.0)

Rainbow Table Generator

Completing my C++ modules in 2nd year of Ethical Hacking I had to make a program that met the requirement of containing self created multithreaded structures. To furfil this I created a Rainbow Table generator that would hash a range of alphanumeric and special characters specified by the user using SHA-1 and save it into a CSV file. For this I received the grade: B (3.0)

Forensics Report

The final module in 2nd year of Ethical Hacking that I enjoyed required that I search for evidence of a crime commited on a disk drive. We were then required to collect our findings into a court report along with how to recreate the steps required to re-aquire all of the evidence discovered. For this I received the grade: B+ (3.5)
